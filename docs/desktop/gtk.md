# GTK Configuration

> [!WARNING]
> This repository has been archived and merged into the main [Caelestia](https://github.com/caelestia-dots/caelestia) repository.

## Overview

GTK3 and GTK4 theming is managed via `adw-gtk3` and custom named colors.

## Installation

Included in the main Caelestia installation.

### Manual Setup

1. Install `adw-gtk-theme`.
2. Apply the theme using `gsettings` or `nwg-look`.
3. Caelestia scripts usually handle color reloading.
