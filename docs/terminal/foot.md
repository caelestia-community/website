# Foot Terminal Configuration

> [!WARNING]
> This repository has been archived and merged into the main [Caelestia](https://github.com/caelestia-dots/caelestia) repository.

## Overview

Foot terminal configuration, primarily for color scheme integration with Caelestia.

## Installation

Included in the main Caelestia dotfiles.

- **Config Path**: `~/.config/foot`

## Configuration

The theme is dynamically referenced in `foot.ini`. The Caelestia scripts update the color scheme file included by the main config.
