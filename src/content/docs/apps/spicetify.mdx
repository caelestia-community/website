# Spicetify Configuration

> [!WARNING]
> This repository has been archived and merged into the main [Caelestia](https://github.com/caelestia-dots/caelestia) repository.

## Overview

Spicetify theme for Spotify.

## Installation

1. Install Spicetify.
2. Symlink the `spicetify` folder from Caelestia dots to `~/.config/spicetify`.
3. Apply config:

```bash
spicetify config current_theme caelestia color_scheme caelestia
spicetify apply
```
