# Fuzzel Configuration

> [!WARNING]
> This repository has been archived and merged into the main [Caelestia](https://github.com/caelestia-dots/caelestia) repository.

## Overview

Configuration for the `fuzzel` application launcher.

## Configuration

The config file is typically located at `~/.config/fuzzel/fuzzel.ini`.
Caelestia uses a background service (or CLI hook) to update the colors in this config file when the wallpaper/theme changes.
