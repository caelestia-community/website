# Fish Shell Configuration

> [!WARNING]
> This repository has been archived and merged into the main [Caelestia](https://github.com/caelestia-dots/caelestia) repository.

## Overview

Fish, Starship, and Fastfetch configurations.

## Installation

Included in the main Caelestia dotfiles.

- **Config Path**: `~/.config/fish`
- **Starship**: `~/.config/starship.toml`
- **Fastfetch**: `~/.config/fastfetch`

## Features

- Custom abbreviations.
- Integrated with `starship` for prompt.
- `fastfetch` branding on startup.
