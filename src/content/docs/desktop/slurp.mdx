# Slurp Configuration

> [!WARNING]
> This repository has been archived and merged into the main [Caelestia](https://github.com/caelestia-dots/caelestia) repository.

## Overview

Provides color theming for `slurp` (screen selection tool).

## Usage

The Caelestia CLI or Shell uses `slurp` for screenshots. The colors are dynamically updated to match the system theme.

### Command Line

When running slurp manually, you might see arguments injected by `caelestia-cli` to apply the border and selection colors.
