# Btop Configuration

> [!WARNING]
> This repository has been archived and merged into the main [Caelestia](https://github.com/caelestia-dots/caelestia) repository.

## Overview

Btop resource monitor theme integration.

## Installation

Included in the main Caelestia dotfiles.

- **Config Path**: `~/.config/btop`

## Usage

Run `btop` in the terminal. The colors should match the active system theme.
